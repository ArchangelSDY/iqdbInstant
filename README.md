imageInstant
===================
Instant search on a pack of image-based search engines.

Supported sites
-------------------
* iqdb.org
* sauceNAO.com
* Google Image
* TinEye
* Baidu Image

*NO IE6 SUPPORT.*

Instruction
-------------------
Go to the [project page]("http://archangelsdy.github.com/iqdbInstant/") for more details.
